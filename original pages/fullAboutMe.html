Crystal Collector

Crystal Collector is a web-based game where the user has to guess the number the computer has randomly selected by clicking the correct gem's to add up to the total number. Each new game loads a new random computer guess (19-120) and each gem has a randomly selected value (1-12). If you add up the gem's to reach the computer's guess, you win! If you go over, you lose!


Technologies used: HTML, CSS, Bootstrap, Javascript and jQuery.

Train Schedule

Train Schedule is an interactive train scheduler that allows you to add new trains and view train runtime in real time. Train Schedule will show you what time the next train arrives, how many minutes away it is and how frequent the train arrives.


Technologies used: HTML, CSS, Bootstrap, Javascript, jQuery, momentJS and Firebase.

Auto-Giphy!

Auto-Giphy! is a automotive themed gif generator. There are 6 pre-made buttons and an area that allows you to add a searchable button. When a gif is searched, you will be presented with 10 gifs related to your search. The rating will also be displayed beneath each gif.


Technologies used: HTML, CSS, Bootstrap, Javascript, jQuery, and GIPHY API.

Liri-Bot

LIRI-Bot is like iPhone's SIRI, however, it is a Language Interpretation and Recognition Interface and is a command-line based app. LIRI will take in your parameters in the command line and you gave back data.


Each command must start with "node liri.js" and is followed by "spotify-this-song 'song name'", "movie-this 'movie name'", "concert-this 'artist name'" or "do-what-it-says". Liri will then load the corresponding results.


Technologies used: Javascript, Node.js (Axios, Moment.js, fileshare), Spotify API, OMDB API and BandsInTown API.

bamazon!

An Amazon-like simulated storefront using the command line to interact with the customer or manager. The user will choose their product and view the final price. The database will then actively update the inventory.


The manager app will ask the manager to view products, view low inventory, add inventory to any current products and add a new product. Manager can go back and forth between options based on what they'd like to do.


Technologies used: Javascript, Node.js (Inquirer, MySQL, cli-table3), MySQL
